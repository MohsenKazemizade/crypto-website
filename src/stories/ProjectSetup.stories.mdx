# 📖 Project Setup Guide

## 1️⃣ Initial Setup with Vite

To create the project with **Vite**, we used the following command:

```sh
npm create vite@latest crypto-website --template react
```

## 2️⃣ Installing Dependencies

The following dependencies were installed:

- **React Router** for navigation:
  ```sh
  npm install react-router-dom
  ```
- **Ant Design** for UI components:
  ```sh
  npm install antd @ant-design/icons
  ```
- **Axios** for API requests:
  ```sh
  npm install axios
  ```

## 3️⃣ Storybook Installation

Storybook was installed using:

```sh
npx sb init
```

To start Storybook, run:

```sh
npm run storybook
```
